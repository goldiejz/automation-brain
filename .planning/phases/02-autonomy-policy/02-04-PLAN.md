<objective>
Replace inline cascading dispatcher logic in `execute-phase.sh::dispatch_task` with a single `policy_dispatcher_route` call. Replace the BLACK-tier hard-stop with a policy_budget_decision call. Make every routing decision audit-logged.

This is the script that actually shipped the manual gate that triggered Phase 2 (the `err "🛑 Budget tier BLACK — refusing to dispatch. Run: ark budget --reset"` line). After this plan, that gate is gone.

**Token-cost masking guard (B-2 fix):** The claude-session branch returns exit 2 without invoking the existing token-record block (which only fires after codex/gemini success). Under Tier 8 with quota stubs, every dispatch hits session, so the budget-tracking would silently masked. This plan adds an explicit prompt-length sentinel cost on the session branch so Phase 3's observer-learner sees representative cost data rather than zeros.

**NEW-W-3 fix:** The session-branch sentinel record uses `|| true` to swallow errors. If the `--record` signature is wrong, recording silently no-ops and the B-2 guarantee is defeated with no test catching it. Task 2a `done` now requires a synthetic dispatch_task run that confirms the budget log gained EXACTLY ONE new line containing `claude-session-handoff` (proving the call ran AND succeeded — not just that the call site exists).
</objective>
